Parametros de Denavit-Hartenberg

Com um posicionamento de sistema de coordenadas eficiente, os parâmetros DH podem ser determinados. Esses parâmetros são determinados seguindo uma série de etapas para compor uma tabela de valores específicos para o sistema de corpo rígido. Resumindo o método usado por Hayat em 2013, as etapas são as seguintes para quaisquer duas frames:

  1. Determinar a rotação do eixo z do quadro i-1 para o quadro i.
  2. Determinar o deslocamento do eixo z da estrutura i-1 para a estrutura i.
  3. Determinar a rotação do eixo x do quadro i-1 para o quadro i.
  4. Determinar o deslocamento do eixo x da estrutura i-1 para a estrutura i.

Realizando este procedimento do frame i ao frame n, resulta numa tabela de parâmetros DH. Os parâmetros DH para o braço robótico Sawyer mostrados na figura 4 são dados na tabela da figura 6 para a posição inicial qz.

Modelagem

Obtendo parametros referidos na figura 4, já conseguimos criar o nosso modelo Sawyer.
Foi utilizada a função Link para a criação da juntas de rotação, e também criado uma variavél qr com uma nova posição inicial.

Fig.5- Tabela de descrição concisa do robot criado

Fig.6- Planar 2 na posição qr que foi criado

Script CriarModeloPlanar2.m

qz = [0 0]; % Posição inicial do Braço
qr = [0 pi/2]; % Outra Posição inicia

L(1) = Link([0 0 1 0]); %Primeira junta de rotação
L(2) = Link([0 0 1 0]); %Primeira junta de rotação
p2 = SerialLink(L, 'name', 'P2','manufacturer','UAL'); % Criar o robo com o nome P2

p2.plot(qr);
