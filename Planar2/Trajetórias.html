Trajetórias

Trajetória

Neste exemplo vamos criar a simulação do Planar 2 a escrever uma letra B (tal como foi feito na cinematica), mas aplicando apenas as trajetórias.

O primeiro problema que se põem, é que não se trabalha com o end-effector (posição do ponto para onde queremos que o robo ponha o seu punho), mas sim com as rotações das juntas.  

Então primeiro é preciso saber o que cada junta precisa de rodar para ter o end-effector desejado.
Para isso podemos utilizar o método teach e assim criamos uma tabela com esses valores:

Encontrado os valores da rotação para cada junta em cada posição, ou seja definir o ponto 1 e 2, que será o trajecto que o robo terá de fazer, e por ai adiante.

Foi aplicado um tempo de 2 segundos para o desenho de cada traço e associado a uma variável t, que foi necessária para definir o trajeto entre ponto com a função jtraj, que calcula o movimento do robor entre os dois ponto defenidos.

  • jtraj(posicao1, posicao2, t);

Então com tudo já defenido, aplicamos um ciclo de repetição por cada traço (trajeto), para percorrer todo o trajeto, e pintar o mesmo a vermelho.

Posição angular

Nestes gráficos podemos analizar as posições de cada junta de rotação como variaram no tempo, por cada traço desenhado.

Script GraphTrajPosJuntaP2All.m

mdl_planar2 %Load do modelo planar 2
% Cordenadas da base (Espaço cartesiano)
p2.base=transl(0,-1.5,0);

% Cordenadas de rotação das juntas para a Letra B
posicao1=[0.5236 2.0944];
posicao2=[0.8481 1.4455];
posicao3=[0.6734 1.4];
posicao4=[0.5035 1.6810];
posicao5=[0.5267 1.7709];
posicao6=[0.6751 1.7913];
posicao7=[0.4218 1.8080];
posicao8=[0.2578 2.0432];
posicao9=[0.5236 2.0944];

t=[0:0.1:2]; % tempo vai de 0 a 2 com incrementação de 0.1

mov = jtraj(posicao1, posicao2, t);
figure(1)
for i=1:1:2 % 2 ciclos = 2 juntas de rotação
    subplot(2,1,i)
    plot(t,mov(:,i)); % Gráfico da posição angular da junta i variando no tempo
    title('Traço 1');
    xlabel('Tempo (s)');
    ylabel(strcat({'Junta '},string(i),{' (rad)'}));
end
                                 .

figure(8)
for i=1:1:2 % 2 ciclos = 2 juntas de rotação

Gráficos de Posição, Velocidade e Acelaração Angulares

Neste gráfico vamos ver como é o comportamento de todas as juntas de rotação, na sua acelaração, velocidade e posição angular, por cada traço desenhado

Script GraphTranjTotalPorTracoP2.m

mdl_planar2 %Load do modelo planar 2
% Cordenadas da base (Espaço cartesiano)
p2.base=transl(0,-1.5,0);

% Cordenadas de rotação das juntas para a Letra B
posicao1=[0.5236 2.0944];
posicao2=[0.8481 1.4455];
posicao3=[0.6734 1.4];
posicao4=[0.5035 1.6810];
posicao5=[0.5267 1.7709];
posicao6=[0.6751 1.7913];
posicao7=[0.4218 1.8080];
posicao8=[0.2578 2.0432];
posicao9=[0.5236 2.0944];

t=[0:0.1:2]; % tempo vai de 0 a 2 com incrementação de 0.1

%Traço 1
mov = jtraj(posicao1, posicao2, t);
H = fkine(p2,mov); % Matriz transformação homogênea
Hmat= H.double; % Converte SE3 para Matriz

%desenha os gráficos posição angular, velocidade angular e acelaração angular - Eixo dos x
figure(1);
subplot(3,1,1)
s = 'Gráfico da coordenada x variando no tempo (Traço 1)';
plot(t,squeeze(Hmat(1,1,:)));
xlabel('Tempo (s)');
ylabel('x (m)');
title(s);

%desenha os gráficos posição angular, velocidade angular e acelaração angular - Eixo dos y
subplot(3,1,2)
s = 'Gráfico da coordenada y variando no tempo (Traço 1)';
plot(t,squeeze(Hmat(2,1,:)));
xlabel('Tempo (s)');
ylabel('y (m)');
title(s);

%desenha os gráficos posição angular, velocidade angular e acelaração angular - Eixo dos x e y
subplot(3,1,3)
s = 'Gráfico das coordenadas x e y (Traço 1)';
plot(squeeze(Hmat(1,1,:)),squeeze(Hmat(2,1,:)));
xlabel('x (m)');
ylabel('y (m)');
title(s);
                                 .
                                 .
                                 .

%Traço 8
mov = jtraj(posicao8, posicao9, t);

figure(8);

s = 'Gráfico da coordenada x variando no tempo (Traço 8)';
plot(t,squeeze(Hmat(1,1,:)));

ylabel('x (m)');
title(s);

xlabel('Tempo (s)');
ylabel('y (m)');
title(s);
