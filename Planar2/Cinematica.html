Cinemática Direta e Inversa

Neste exemplo vamos criar a simulação do Planar 2 a escrever uma letra .

Vamos utilizar cinemática do robô, que se interessa em descrever o movimento de um corpo controlável com vários graus de liberdade e referenciais, estabelecendo as relações necessárias para a migração entre eles. Dentro dele, a Cinemática direta e a Cinemática inversa são subdivididas. O primeiro gera as expressões que permitem ir do espaço das coordenadas da junta do robô ao espaço das coordenadas da extremidade do robô, o segundo trabalha na direção oposta, partindo das coordenadas do final para determinar o valor a ser tomado por cada uma das articulações do braço.

Foi utilizada a função ctraj para defenir o end-Effector entre as dua posições (pp1-> 1ª posição e pp2 -> 2ª posição), para cada traço do robo e onde foi considerado 20 passos para esta ação.

  • pp1 = transl(0,-0.5,0); 
  • pp2 = transl(0,0,0); 
  • crt_sp1 = ctraj(pp1,pp2,20); 

Definda as posições do end-effector (crt_sp1), vamos utilizar a cinematica inversa atravéz da função ikine, onde vamos utilizar a mascara (mask) [1 1 0 0 0 0], para apenas encontrar o end-effector dos eixos x e y, pois não vamos trabalhar em mais nenhum eixo.

  • p2.ikine(crt_sp1,'mask',[1 1 0 0 0 0]);

Para desenhar as linhas criamos um ciclo for onde aplicamos a cinematica direta, através da função fkine, tranformando a pose encontrada pelo metodo ikine.

  • fd1 = p2.fkine(crt_ik1(i,:))

E utilizamos a matriz transposta da posição angular dada pela cinemática direta, para o desenho do traço.

  • j = transpose(fd1.t);

Foram precisos 9 ciclos (traços) para escrever a letra B

mdl_planar2 %Load do modelo planar 2
% Cordenadas da base (Espaço cartesiano)
p2.base=transl(0,-1.5,0);
po = 20; % numero de passo para desenhar cada traço
% Cordenadas para desenhar a Letra B
% Ponto 1 a 2
pp1 = transl(0,-0.5,0); % posição 1
pp2 = transl(0,0,0); % posição 1
crt_sp1 = ctraj(pp1,pp2,po); % defenir o end-effector das posições
crt_ik1 = p2.ikine(crt_sp1,'mask',[1 1 0 0 0 0]); % Utilização da mascara para apenas encontrar o End-effector do x e y 
                                 .

% Ponto 9 a 10
pp10 = transl(0,-0.5,0);
crt_sp9 = ctraj(pp9,pp10,po);
crt_ik9 = p2.ikine(crt_sp9,'mask',[1 1 0 0 0 0]);

t=[0.1:0.1:2];
% desenha os varios grafico de posição de cada traço
figure(1)
for i=1:1:2 % 2 ciclos = 2 juntas de rotação
