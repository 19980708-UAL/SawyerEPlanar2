Trajetórias 

Introdução

Neste exemplo também vamos utilizar a introdução de um objecto (copo) e uma superficie (mesa), e criar a mesma simulação feita na cinemática, defenindo uma trajetória final.

Como podemos ver(Figura 2), o Robot Sawyer não precisa de muito espaço para operar.

Na figura 3 vemos as posições das juntas por tempo, ou seja em que posição a junta se encontrava em cada 0.05 segundos

O primeiro problema que se põem, é que não se trabalha com o end-effector (posição do ponto para onde queremos que o robo ponha o seu punho), mas sim com as rotações das juntas.  

Então primeiro é preciso saber o que cada junta precisa de rodar para ter o end-effector desejado.
Para isso podemos utilizar o método teach e assim criamos uma tabela com esses valores: 

Encontrado os valores da rotação para cada junta em cada posição, ou seja definir o posição inicial ao posição final, que será o trajecto que o robo terá de fazer.

Foi aplicado um tempo de 2 segundo para o desenho de cada traço e associado a uma variável t, que foi necessária para definir o trajeto entre ponto com a função jtraj, que calcula o movimento do robor entre os dois ponto defenidos.

  • jtraj(posicao1, posicao2, t)

Então com tudo defenido, é so fazer plot do movimento gerado, e o Sawyer já vai até aso copo.

Posição

Como se pode ver nem todas as juntas precisaram de fazer a rotação, como mostra o gráfico da figura 4, que a 4ª e a 6ª junta não se mexeram, ao contrário do que acontece com a cinematica direta e inversa

Fig.4- Gráfico da posição angular das juntas(1-7) variando no tempo

Script  GraphPosJunta.m

mdl_sawyer %Load do Modelo Sawyer

t = [0:.05:2]; % Tempo da transição: 1 a 2 com incrementação de 0.5

posicao1 = [0, -pi/2, 0, 0, 0, 0, 0] % Posição angular inicial (radianos) do Robot Sawyer
posicao2 = [0.4, 0, -pi/2, 0, pi/2-0.2, 0, -0.3] % Posição angular s (radianos) do Robot Sawyer

mov = jtraj(posicao1, posicao2, t); % Trajetória das coordenadas das juntas

figure(1);

for i=1:1:7
    subplot(7,1,i)
    plot(t,mov(:,i)); % Gráfico da posição angular da junta i variando no tempo
    title('Theta');
    xlabel('Tempo (s)');
    ylabel(strcat({'Junta '},string(i),{' (rad)'}));
end;

Script GraphTranjporJunta.m

mdl_sawyer %Load do Modelo Sawyer

t = [0:.05:2]; % Tempo da transição: 1 a 2 com incrementação de 0.5

posicao1 = [0, -pi/2, 0, 0, 0, 0, 0] % Posição angular inicial (radianos) do Robot Sawyer
posicao2 = [0.4, 0, -pi/2, 0, pi/2-0.2, 0, -0.3] % Posição angular final (radianos) do Robot Sawyer

[q, dq, ddq] = jtraj(posicao1, posicao2, t); % Trajetória das coordenadas das juntas

for i=1:1:7
    figure(i);

    subplot(3,1,1)
    plot(t,q(:,i)); % Gráfico da posição angula r
    title('Theta');
    xlabel('Tempo(s)');
    ylabel(strcat({'Junta '},string(i),{' (rad)'}));

    subplot(3,1,2)
    plot(t,dq(:,i)); % Gráfico da velocidade angular
    title('Velocidade')
    xlabel('Tempo (s)');
    ylabel(strcat({'Junta '},string(i),{' vel(rad/s)'}))

    subplot(3,1,3)
    plot(t,ddq(:,i)); % Gráfico da aceleração angular
    title('Aceleração')
    xlabel('Tempo (s)');
    ylabel(strcat({'Junta '},string(i),{' aceler(rad/s2)'}))
end

Script GraphTranjTotal.m

mdl_sawyer %Load do Modelo Sawyer

t = [0:.05:2]; % Tempo da transição: 1 a 2 com incrementação de 0.5

posicao1 = [0, -pi/2, 0, 0, 0, 0, 0] % Posição angular inicial (radianos) do Robot Sawyer
posicao2 = [0.4, 0, -pi/2, 0, pi/2-0.2, 0, -0.3] % Posição angular final (radianos) do Robot Sawyer

mov = jtraj(posicao1, posicao2, t); % Trajetória das coordenadas das juntas

H = fkine(sawyer,mov); % Matriz transformação homogênea
Hmat= H.double; % Converte SE3 para Matriz
figure(1);
subplot(3,1,1)
s = 'Gráfico da coordenada x variando no tempo';
plot(t,squeeze(Hmat(1,1,:)));
xlabel('Tempo (s)');
ylabel('x (m)');
title(s);
subplot(3,1,2)
s = 'Gráfico da coordenada y variando no tempo';
plot(t,squeeze(Hmat(2,1,:)));
xlabel('Tempo (s)');
ylabel('y (m)');
title(s);
subplot(3,1,3)
s = 'Gráfico da coordenada z variando no tempo';
plot(t,squeeze(Hmat(3,1,:)));
xlabel('Tempo (s)');
ylabel('z (m)');
title(s);

figure(2);
s = 'Gráfico das coordenadas x,y e z';
plot3(squeeze(Hmat(1,1,:)),squeeze(Hmat(2,1,:)),squeeze(Hmat(3,1,:)));
xlabel('x (m)');
ylabel('y (m)');
zlabel('z (m)');
title(s);
